# Fraud Detection Lab

Credit Card Fraud Detection using XGBoost + SMOTE deployed on Amazon SageMaker.

## Stack
- Amazon SageMaker (Training + Endpoints)
- XGBoost (Base + SMOTE)
- Random Cut Forest (Anomaly Detection)
- Gradio (Demo UI)
- Python 3

## Model Performance
| Metric | SMOTE Model |
|--------|-------------|
| Precision | 93% |
| Recall | 80% |
| F1 Score | 86% |
